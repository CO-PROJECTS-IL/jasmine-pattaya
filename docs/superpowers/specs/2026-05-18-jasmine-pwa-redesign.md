# יסמין PWA — מסמך עיצוב (עיצוב מחדש)

## סקירה כללית

אפליקציית PWA למסעדת "יסמין" — מסעדה ישראלית בפאטייה, תאילנד. האפליקציה משרתת 3 סוגי משתמשים דרך פורטלים נפרדים: לקוחות, עובדים, ומנהל.

### סטאק טכנולוגי
- React 19 + Vite + TypeScript
- Tailwind CSS 4
- Supabase (PostgreSQL + Realtime + Edge Functions)
- Cloudinary (תמונות)
- GitHub Pages (אירוח)
- HashRouter (ניתוב)

### עיצוב
- **סגנון:** יוקרה כהה
- **צבעים ראשיים:** שחור (#080808, #121212, #1a1a1a) + זהב (#c9a84c, #d4b96a, #a88a3a)
- **פונטים:** Heebo + Frank Ruhl Libre (עברית), פונט מערכת (תאית/אנגלית)
- **RTL/LTR:** מתחלף לפי שפה

### שפות
- עברית (RTL), אנגלית (LTR), תאית (LTR)
- מתג שפה נגיש בכל מסך
- ברירת מחדל לפי פורטל: לקוחות=עברית, עובדים=תאית, מנהל=עברית

## פורטל לקוחות

### כניסה
- מסך כניסה עם לוגו "יסמין" ותת-כותרת
- הזנת מספר שולחן או סריקת QR
- אייקון קטן חצי-שקוף בפינה — לחיצה מעבירה למסך כניסת צוות
- מספר שולחנות דינמי (מוגדר על ידי המנהל)

### ניווט תחתון
2 טאבים בלבד:
1. **תפריט** — תפריט המנות הרגיל
2. **ארוחת שישי** — הזמנה מראש לארוחת שישי

### תפריט
- קטגוריות עם גלילה אופקית
- כרטיסי מנות: תמונה, שם (3 שפות), תיאור, מחיר (בבאט ฿), תגיות (כשר/חריף/צמחוני)
- לחיצה על מנה → מודאל פרטים עם בחירת כמות + "הוסף להזמנה"
- כפתור עגלה צף עם מספר פריטים

### מעבר אוטומטי לתפריט שישי
- ביום שישי, משעה שהמנהל מגדיר בהגדרות — התפריט הרגיל מוחלף **לחלוטין** בתפריט שישי
- הלקוח רואה רק את תפריט שישי, אין גישה לתפריט הרגיל
- המעבר אוטומטי לפי שעה + יום בשבוע (שעון תאילנד, UTC+7)

### עגלה
- סליידר תחתון (bottom drawer)
- כמויות +/-, הסרת פריט, שדה הערות
- מספר שולחן מוצג
- סכום כולל
- כפתור "שלח הזמנה"

### אישור הזמנה
- מסך אישור עם מספר הזמנה
- **סכום כולל + הכיתוב "לא כולל שירות"**
- כפתור "הזמנה חדשה"

### ארוחת שישי (הזמנה מראש)
- רשימת מנות מתפריט שישי מיוחד עם **מחיר לכל מנה** (בחירה חופשית)
- טופס: שם, טלפון, מספר אורחים, בחירת מנות
- סכום כולל מחושב
- ארוחת שישי רצה **אוטומטית כל שבוע** — לא צריך ליצור כל פעם

### תשלום
- תשלום פיזי בלבד (מזומן/כרטיס למלצר)
- המלצר/מנהל מסמן "שולם" בדשבורד ההזמנות

## פורטל עובדים

### כניסה
- מסך כניסת צוות (מהאייקון בפינה)
- כפתור "เข้าสู่ระบบพนักงาน" (כניסת עובדים — בתאית)
- PIN 4 ספרות, ברירת מחדל: 0000
- ברירת מחדל שפה: תאית

### הגבלת מיקום (GPS)
- **כל מסך העובדים** עובד רק בטווח 50 מטר מהמסעדה
- בדיקת מיקום באמצעות Geolocation API
- אם העובד מחוץ לטווח — מסך חסימה עם הודעה
- מיקום המסעדה מוגדר דינמית על ידי המנהל
- **עקיפת GPS:** המנהל יכול לאשר צ'ק-אין ידני לעובד. מסומן בדוח כ"ידני"

### שעון נוכחות
- כפתור גדול "เข้างาน" (התחל משמרת) / "ออกงาน" (סיים משמרת)
- טיימר שמראה כמה זמן במשמרת
- שעון לפי אזור זמן תאילנד (UTC+7)

### צבירת שכר
- העובד רואה כמה כסף צבר **מתחילת החודש**
- חישוב: שעות/ימים × שכר שעתי/יומי
- **המנהל יכול להשבית** תצוגה זו בהגדרות

### הזמנה חדשה
- כפתור "สั่งอาหารใหม่" (הזמנה חדשה)
- לחיצה → בחירת מספר שולחן → תפריט → בחירת מנות → שליחה
- **ביום שישי** — הכפתור מוביל לתפריט שישי
- עדכון בזמן אמת (Realtime) — הזמנה מופיעה בדשבורד מיד

### דשבורד הזמנות (קנבן)
- **4 עמודות צבעוניות** זו-לצד-זו:
  - 🔴 **חדשה** (אדום) — הזמנה שנכנסה עכשיו
  - 🟠 **בהכנה** (כתום) — המטבח מכין
  - 🟡 **הוגשה** (צהוב) — הגיע לשולחן
  - 🟢 **שולמה** (ירוק) — הלקוח שילם
- כל הזמנה = כרטיס עם: מספר שולחן, פריטים, הערות, זמן
- **שינוי סטטוס קדימה ואחורה** — אפשר להחזיר הזמנה לסטטוס קודם מכל שלב
- **התראה צלילית** על הזמנה חדשה + הבהוב
- **עדכון בזמן אמת** (Supabase Realtime)

### תכנית עבודה
- העובד רואה את המשמרות שלו (קריאה בלבד)
- תצוגת שבוע נוכחי + שבוע הבא

## פורטל מנהל

### כניסה
- מסך כניסת צוות (מהאייקון בפינה)
- כפתור "כניסת מנהל" (בעברית)
- PIN 4 ספרות, ברירת מחדל: 1111
- ברירת מחדל שפה: עברית

### מסך בית
8 כפתורים גדולים בתצוגת רשת:
1. 📊 **דשבורד**
2. 🍽️ **הזמנות**
3. 📋 **ניהול תפריט**
4. 👥 **ניהול עובדים**
5. 📅 **תכנית עבודה**
6. 🕯️ **ארוחת שישי**
7. 💰 **דוחות כספיים**
8. ⚙️ **הגדרות**

כל כפתור פותח מסך נפרד עם כפתור "חזרה" למסך הבית.

### 1. דשבורד
- כרטיסי סיכום: הזמנות היום, הכנסות היום, הזמנה ממוצעת
- הזמנות לפי סטטוס (ספירה)
- גרף הזמנות — התפלגות שעתית היום

### 2. הזמנות
- דשבורד קנבן זהה למסך העובדים
- 4 עמודות צבעוניות עם שינוי סטטוס דו-כיווני
- **סימון "שולם"** — רק מנהל/עובד (לא הלקוח)

### 3. ניהול תפריט
- רשימת מנות לפי קטגוריות
- הוספה / עריכה / מחיקת מנה
- לכל מנה: שם (3 שפות), תיאור (3 שפות), מחיר (באט), תמונה (Cloudinary), קטגוריה
- תגיות: כשר, חריף, צמחוני
- מתג זמינות (פעיל/לא פעיל)
- ניהול קטגוריות: הוספה/עריכה/מחיקה/סדר

### 4. ניהול עובדים
- **תצוגת כרטיסים** — כל עובד ככרטיס
- פרטי עובד:
  - שם מלא
  - טלפון + כפתור WhatsApp + כפתור חיוג
  - תמונת פרופיל
  - תפקיד (טבח / מלצר / ברמן / אחר)
  - סוג שכר (שעתי / יומי)
  - סכום שכר
  - תאריך התחלה
  - הערות
  - מסמכים (דרכון / ויזה — העלאת תמונה)
  - ימי חופש
  - סטטוס (פעיל / לא פעיל)
- היסטוריית נוכחות של כל עובד
- סיכום שכר חודשי
- **עריכת שעות נוכחות** — המנהל יכול לתקן רשומות
- **צ'ק-אין ידני** — עקיפת GPS, מסומן כ"ידני" בדוח
- הוספה / עריכה / השבתת עובד

### 5. תכנית עבודה
- **טבלה שבועית**: שורה לכל עובד, עמודה לכל יום (ראשון-שבת)
- לחיצה על תא → בחירת משמרת: בוקר / ערב / יום מלא / חופש / מותאם
- **שעות משמרת מוגדרות** על ידי המנהל (למשל: בוקר = 10:00-16:00, ערב = 17:00-23:00)
- אפשרות להגדיר שעות מותאמות לתא ספציפי
- **העתקת שבוע** — העתקת לוח שבוע קודם ועריכה
- העובדים רואים את המשמרות שלהם במסך העובד

### 6. ארוחת שישי
- **רצה אוטומטית כל שבוע** — לא צריך ליצור כל פעם
- המנהל מגדיר:
  - תפריט שישי מיוחד (בחירת מנות + מחירים ייעודיים)
  - מקסימום אורחים (אופציונלי)
  - שעת מעבר — מאיזו שעה ביום שישי התפריט הרגיל מוחלף בתפריט שישי
- **ביטול שישי ספציפי** — המנהל יכול לבטל ארוחת שישי לשבוע מסוים
- צפייה בהזמנות מראש: רשימת מזמינים עם שם, טלפון, מספר אורחים, מנות שנבחרו
- אישור / ביטול הזמנות

### 7. דוחות כספיים
- **דשבורד מפורט** עם מספר גרפים:
  - כרטיסי סיכום: הכנסות, הוצאות, רווח נקי
  - גרף הכנסות לאורך זמן (עמודות)
  - גרף הוצאות לפי קטגוריה (עוגה)
  - גרף רווח לאורך זמן (קו)
  - פירוט עלות עובדים
- **סינון לפי תאריכים**: יום, שבוע, חודש, טווח מותאם
- **3 סוגי הוצאות:**

#### 7.1 עלות עובדים (אוטומטי)
- חישוב אוטומטי: שעות נוכחות × שכר שעתי/יומי
- פירוט לפי עובד
- סיכום חודשי

#### 7.2 הוצאות חוזרות
- שם הוצאה (למשל: שכירות, חשמל, מים, ביטוח)
- סכום
- תדירות: יומי / שבועי / חודשי / שנתי
- תאריך התחלה
- המערכת מחשבת אוטומטית לכל תקופת דוח

#### 7.3 הוצאות חד-פעמיות
- שם, סכום, תאריך
- קטגוריה (אופציונלי): ציוד, תיקון, סחורה, אחר

### 8. הגדרות
- **PIN מנהל** — שינוי קוד 4 ספרות
- **PIN עובדים** — שינוי קוד 4 ספרות
- **מיקום מסעדה** — הגדרת קואורדינטות GPS (כפתור "הגדר מיקום נוכחי" כשנמצא במסעדה)
- **מספר שולחנות** — דינמי, משנה את כמות ה-QR codes
- **QR codes** — יצירה והדפסה לכל שולחן
- **שעות פתיחה** — לפי יום בשבוע
- **שעות משמרות** — הגדרת שעות בוקר/ערב/מותאם
- **שעת מעבר תפריט שישי** — מאיזו שעה ביום שישי מוחלף התפריט
- **תצוגת שכר לעובדים** — הפעלה/השבתה

## מבנה טכני

### ניתוב (HashRouter)
```
# לקוחות
/                     → מסך כניסה (מספר שולחן)
/menu                 → תפריט (או תפריט שישי אם רלוונטי)
/cart                 → עגלה
/order-confirmation   → אישור הזמנה
/friday-dinner        → הזמנה מראש לארוחת שישי

# כניסת צוות
/staff                → מסך PIN (מנהל + עובדים)

# עובדים
/employee             → מסך עובדים ראשי (צ'ק-אין + דשבורד)
/employee/new-order   → הזמנה חדשה (תפריט)
/employee/schedule    → תכנית עבודה

# מנהל
/admin                → מסך בית (8 כפתורים)
/admin/dashboard      → דשבורד
/admin/orders         → הזמנות (קנבן)
/admin/menu           → ניהול תפריט
/admin/menu/edit/:id  → עריכת מנה
/admin/employees      → ניהול עובדים
/admin/employees/:id  → פרטי עובד
/admin/schedule       → תכנית עבודה
/admin/friday         → ארוחת שישי
/admin/reports        → דוחות כספיים
/admin/settings       → הגדרות
/admin/qr-codes       → QR codes
```

```
admin-reports         → נתוני דוחות כספיים עם סינון (מנהל)
admin-settings        → עדכון הגדרות (מנהל)
public-settings       → שליפת הגדרות ציבוריות (כולם)
```

### Realtime
- טבלת `orders`: INSERT + UPDATE → דשבורד הזמנות מתעדכן בזמן אמת
- התראה צלילית (Web Audio API) על הזמנה חדשה

### PWA
- Service Worker עם Workbox
- Precache: סטטיים (JS, CSS, פונטים, אייקונים)
- Runtime cache: Supabase API (StaleWhileRevalidate), Cloudinary (CacheFirst, 7 ימים)
- Manifest: שם, אייקונים, צבעי ערכת נושא, תצוגה standalone

### אזור זמן
- כל הזמנים באפליקציה — **שעון תאילנד (UTC+7)**
- צ'ק-אין/אאוט, שעות משמרת, שעת מעבר תפריט שישי — הכל לפי Asia/Bangkok
